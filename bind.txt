Ctrl + Escape — открыть/закрыть терминал (если фокус на терминале)

Cmd + B — перейти к определению символа

A (в проводнике) — создать новый файл

Shift + A (в проводнике) — создать новую папку

Alt + Enter (в редакторе) — быстрое исправление

Alt + Enter (в терминале) — быстрое исправление (если поддерживается)

Alt + Enter (в панели проблем) — показать варианты исправления

Cmd + Shift + E (если проводник закрыт) — открыть проводник

Cmd + Shift + E (если открыт) — скрыть боковую панель

Cmd + K Cmd + B — удалён биндинг скрытия панели

Shift + Cmd + Ctrl + E — фокус на проводнике

Shift + Cmd + Ctrl + S — показать/скрыть статусбар

Ctrl + Tab — следующий редактор

Shift + Cmd + ] — удалён биндинг следующего редактора

Ctrl + Shift + Tab — предыдущий редактор

Shift + Cmd + [ — удалён биндинг предыдущего редактора

PageUp PageUp — переименовать символ

Delete Delete — удалить строку

Cmd + . — открыть и сфокусировать хлебные крошки (breadcrumbs)

Shift + Cmd + G — перейти в Source Control (SCM)

Ctrl + Shift + G — удалён биндинг SCM

Shift + Cmd + B — перейти к определению типа

Shift + Cmd + C — свернуть все списки

Shift + Cmd + ↑ — удалён биндинг сворачивания списка

Cmd + B — удалён биндинг скрытия боковой панели
